# Сравнение автокодировщиков для Fashion-MNIST

Проект по исследованию и сравнению различных архитектур автокодировщиков на датасете Fashion-MNIST. Реализованы и проанализированы три типа автокодировщиков: базовый, разреженный и вариационный (VAE).

## О проекте

Этот проект представляет собой комплексное исследование автокодировщиков для задачи сжатия и реконструкции изображений одежды. Включает:

- **Три архитектуры автокодировщиков**:
  -  Базовый сверточный автокодировщик
  -  Разреженный автокодировщик с L1-регуляризацией
  -  Вариационный автокодировщик (VAE) с KL-дивергенцией

- **Полный анализ**:
  -  Кривые обучения всех моделей
  -  Визуализация реконструкций
  -  Анализ латентных пространств
  -  Интерполяция в латентном пространстве
  -  Анализ эффективности сжатия
  -  Интерактивная демонстрация
